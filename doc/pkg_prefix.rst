Package identifier
==================

Prefices or identifiers of package-specific scripts
and workflows are listed below.
The identifier is a mnemonic for the package and
no more than 4-letter long.

+------------------+------------+
| Package          | Identifier |
+==================+============+
| VASP             | ``vasp``   |
| WIEN2k           | ``w2k``    |
| GPAW             | ``gpaw``   |
| ABINIT           | ``abi``    |
| FHI-aims         | ``aims``   |
| Quantum-ESPRESSO | ``qe``     |
| BerkeleyGW       | ``bgw``    |
| GAP2c            | ``gap``    |
| CP2k             | ``cp2k``   |
+------------------+------------+
